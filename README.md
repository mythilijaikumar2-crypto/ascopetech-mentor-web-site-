# Ascope Tech Mentor Web Application

An AI-powered career mentor platform guiding candidates with personalized resume analysis, interactive mock interviews, study roadmaps, and targeted job recommendations. Built with React, TypeScript, Tailwind CSS, Vite, and Framer Motion.

## Features

- **AI Career Guidance**: Interactive assessment matching candidates with suitable career paths.
- **Resume Builder & ATS Analyzer**: Create and score ATS-friendly resumes.
- **Mock Interviews**: Interactive timed chat simulations with feedback and scoring.
- **Personalized Roadmap**: Step-by-step milestone learning roadmap.
- **Job Recommendations**: Tailored job listings and application tracking.

## Getting Started

### Installation

```bash
npm install
```

### Development Server

```bash
npm run dev
```

### Production Build

```bash
npm run build
```
